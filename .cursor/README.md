# Cursor rules

This folder contains Cursor rules and related configuration for the sempipes-demo project.

- **no-edit-sempipes.mdc** — Do not edit anything under the `sempipes/` symlink.
- **commit-conventions.mdc** — Small, frequent commits with one-line messages.
- **setup-and-run.mdc** — Project setup (Poetry, `pyproject.toml` only), how to run backend/frontend, and tests.
